## 正则表达式操作:基础

正则表达式使用元字符来匹配静态或动态数据字符串。 *meta* 这个词是一个拉丁(希腊语)词，意思是之后的*、*之后的*或者*之后的*。当在编程语言中使用时，元字符从它们的实际表示中具有特殊的含义，因此元字符可以被定义为具有隐藏含义的字符。元字符包括以下内容:*

```py
  . ^ $ * + ? \ | ( ) [ ] { }

```

当在正则表达式中使用这些元字符之一时，它具有一些特殊的含义。最后六个字符成对出现，所以在正则表达式中必须成对使用括号。先从最直白的正则表达式开始，慢慢积累知识。此外，如果您想将这些字符表示为单个文字字符，那么元字符可以用一组方括号(`[ ]`)括起来，或者使用反斜杠(`\`)进行转义。但是，要特别注意元字符`^`和`\`；使用方括号方法无法精确匹配这两个字符。见表 9-1 。
